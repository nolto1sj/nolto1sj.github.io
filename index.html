Boots N' Cats

Boots N' Cats is an album review site that utilizes Spotify's Web API as well as our own API and database. Here, one can search through Spotify's catalog and choose an album to review. This review will then be posted on the reviews page alongside all users' reviews. This app also includes an artist aggregator that generates a web of related artists with links to their Spotify artist page.
